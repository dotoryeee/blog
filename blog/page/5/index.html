Terraform Backend

목표

  • Terraform의 Backend 기능을 이용해 state파일을 안전하게 관리해봅니다.
  • .tfstate를 로컬이 아닌 S3에 저장하고 DynamoDB를 이용해 파일을 Locking합니다

  • Backend를 이용함에 따른 이점

    • Locking

      협업시 원격 저장소를 사용하여 여러명이 동시에 같은 state 접근하는것을 막습니다

    • Backup

      원격저장소에 저장함으로써 데이터 유실을 방지합니다. 특히 S3에 저장하는 방법은 버전관리가 가능한 편리하고 안전한 방법입니다.

Terraform Practice

💡 테라폼을 아키텍처를 연습해봅니다

  • EC2에 SSH 접속을 위한 키페어 생성

    1. 비밀키를 생성합니다
    ssh-keygen -t rsa -b 2048 -f ~/dotoryeee.pem -q -P
    1. pem을 사용하려면 권한을 수정해야 합니다
    chmod 400 ~/dotoryeee.pem
    1. AWS에 넘겨줄 공개키를 생성합니다
    ssh-keygen -y -f ~/dotoryeee.pem > ~/dotoryeee.pub

목표
