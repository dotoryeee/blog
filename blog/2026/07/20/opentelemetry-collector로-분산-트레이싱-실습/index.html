OpenTelemetry CollectorлЎң л¶„мӮ° нҠёл ҲмқҙмӢұ мӢӨмҠө

лӘ©н‘ң

  • Collector нҢҢмқҙн”„лқјмқёмқ„ лЎңм»¬м—җ м„ёмҡ°кі , м„ңлЎң нҳём¶ңн•ҳлҠ” л‘җ м„ңл№„мҠӨмқҳ мҡ”мІӯ н•ҳлӮҳк°Җ м–ҙл–»кІҢ л¶„мӮ° нҠёл ҲмқҙмҠӨлЎң мқҙм–ҙм§ҖлҠ”м§Җ мӢӨмёЎн•ңлӢӨ
  • м• н”ҢлҰ¬мјҖмқҙм…ҳ мҪ”л“ңлҘј кі м№ҳм§Җ м•ҠлҠ” zero-code кі„мёЎмңјлЎң нҢҢмқҙмҚ¬ м„ңл№„мҠӨ л‘җ к°ңлҘј лқ„мҡ°кі , Collectorмқҳ debug exporter лЎңк·ём—җм„ң span мҲҳмӢ мқ„ лҲҲмңјлЎң нҷ•мқён•ңлӢӨ
  • traceparent н—ӨлҚ”к°Җ м„ңл№„мҠӨ кІҪкі„лҘј л„ҳм–ҙ м „лӢ¬лҗҳлҠ” мҰқкұ°лҘј м•ұ лЎңк·ёмҷҖ spanмқҳ л¶ҖлӘЁ кҙҖкі„лЎң көҗм°Ё кІҖмҰқн•ңлӢӨ

Tip

рҹ’Ў CollectorмҷҖ Jaeger, кі„мёЎ лҢҖмғҒ м•ұмқ„ м „л¶Җ docker composeлЎң лЎңм»¬м—җм„ң лҸҢл Ө нҒҙлқјмҡ°л“ң л№„мҡ©мқҙ л“Өм§Җ м•ҠлҠ”лӢӨ

кі„мёЎ к°ңл…җкіј Collector кө¬м„ұмҡ”мҶҢ(receiver, processor, exporter), мӢ нҳё 3мў…, мғҳн”Ңл§Ғ л°©мӢқмқҖ OpenTelemetry м •лҰ¬ кёҖм—җм„ң лӢӨлӨҳлӢӨ. мқҙ кёҖмқҖ к·ё мң„м—җм„ң нҢҢмқҙн”„лқјмқёмқ„ мӢӨм ңлЎң м„ёмҡ°кі , н•ң мҡ”мІӯмқҙ л‘җ м„ңл№„мҠӨлҘј кұ°міҗ н•ҳлӮҳмқҳ traceлЎң л¬¶мқҙлҠ”м§Җ нҷ•мқён•ҳлҠ” лҚ° м§‘мӨ‘н•ңлӢӨ.

мӢӨмҠө кө¬м„ұ

м»Ён…Ңмқҙл„Ҳ л„Ө к°ңлҘј н•ң compose л„ӨнҠёмӣҢнҒ¬м—җ мҳ¬лҰ°лӢӨ. м•ұмқҖ CollectorлЎң OTLPлҘј ліҙлӮҙкі , CollectorлҠ” л°ӣмқҖ traceлҘј JaegerлЎң л„ҳкёҙлӢӨ. нқҗлҰ„мқҖ лӢӨмқҢкіј к°ҷлӢӨ.

  • otel-frontend: 8000 нҸ¬нҠём—җм„ң мҡ”мІӯмқ„ л°ӣм•„ otel-backendлҘј HTTPлЎң нҳём¶ңн•ҳлҠ” нҢҢмқҙмҚ¬ м„ңл№„мҠӨ
  • otel-backend: 9000 нҸ¬нҠём—җм„ң JSONмқ„ лҸҢл ӨмЈјлҠ” нҢҢмқҙмҚ¬ м„ңл№„мҠӨ
  • otel-collector: OTLPлҘј 4317(gRPC)В·4318(HTTP)лЎң л°ӣм•„ к°Җкіөн•ң л’Ө JaegerлЎң export
  • otel-jaeger: Collectorк°Җ ліҙлӮё traceлҘј м ҖмһҘн•ҳкі  16686 UIлЎң мЎ°нҡҢ

мқҙлҜём§ҖлҠ” otel/opentelemetry-collector-contrib:0.109.0, jaegertracing/all-in-one:1.62.0, м•ұмқҖ python:3.12-slimм—җ Flask 3.0.3кіј opentelemetry-distro 0.48b0лЎң кі м •н•ңлӢӨ.

Collector м„Өм •

лЁјм Җ Collector м„Өм • нҢҢмқјмқ„ мһ‘м„ұн•ңлӢӨ. otlp receiverлҘј 4317кіј 4318м—җ м—ҙкі , processorлЎң attributesмҷҖ batchлҘј мҲңм„ңлҢҖлЎң нғңмҡҙ л’Ө, exporter л‘җ к°ң(Jaegerмҡ© otlpмҷҖ мҪҳмҶ” нҷ•мқёмҡ© debug)лЎң лӮҙліҙлӮёлӢӨ.

vi otel-collector-config.yaml
otel-collector-config.yaml
receivers:
  otlp:
    protocols:
      grpc:
        endpoint: 0.0.0.0:4317   #OTLP gRPC мҲҳмӢ 
      http:
        endpoint: 0.0.0.0:4318   #OTLP HTTP мҲҳмӢ 

processors:
  attributes/dotoryeee:          #лӘЁл“  spanм—җ кіөнҶө нғңк·ё мЈјмһ…
    actions:
      - key: dotoryeee.lab
        value: otel-collector-lab
        action: insert
  batch:
    timeout: 1s                  #1мҙҲ лӢЁмң„лЎң л¬¶м–ҙ export

exporters:
  otlp/jaeger:
    endpoint: otel-jaeger:4317   #Jaegerк°Җ 4317м—җм„ң OTLP мҲҳмӢ 
    tls:
      insecure: true
  debug:
    verbosity: detailed          #мҲҳмӢ н•ң spanмқ„ мҪҳмҶ”м—җ к·ёлҢҖлЎң м¶ңл Ҙ

service:
  pipelines:
    traces:
      receivers: [otlp]
      processors: [attributes/dotoryeee, batch]
      exporters: [otlp/jaeger, debug]
  • receiverлҘј 0.0.0.0мңјлЎң л°”мқёл”©н•ҙм•ј лӢӨлҘё м»Ён…Ңмқҙл„Ҳм—җм„ң м ‘к·јн•  мҲҳ мһҲкі , endpointлҘј otel-jaegerлЎң м Ғмңјл©ҙ compose л„ӨнҠёмӣҢнҒ¬к°Җ м»Ён…Ңмқҙл„Ҳ мқҙлҰ„мқ„ DNSлЎң н’Җм–ҙмӨҖлӢӨ
  • debug exporterлҠ” л°ұм—”л“ңлЎң ліҙлӮҙлҠ” кІғкіј лі„к°ңлЎң мҪҳмҶ”м—җ spanмқ„ м°Қм–ҙ, мҲҳмӢ  м—¬л¶ҖлҘј лЎңк·ёл§Ң ліҙкі  нҢҗлӢЁн•  мҲҳ мһҲкІҢ н•ҙмӨҖлӢӨ

Jaeger 1.35 мқҙнӣ„ all-in-oneмқҖ COLLECTOR_OTLP_ENABLEDлЎң 4317В·4318 OTLP мҲҳмӢ мқ„ мјӨ мҲҳ мһҲлӢӨ. мқҙ мӢӨмҠөмқҖ Collectorк°Җ JaegerлЎң лӢӨмӢң OTLPлҘј ліҙлӮҙлҜҖлЎң мқҙ к°’мқ„ нҷңм„ұнҷ”н•ңлӢӨ.

кі„мёЎ лҢҖмғҒ м•ұ

л‘җ м„ңл№„мҠӨлҠ” нҸүлІ”н•ң Flask м•ұмқҙлӢӨ. кі„мёЎ мҪ”л“ңлҠ” н•ң мӨ„лҸ„ л„Јм§Җ м•ҠлҠ”лӢӨ. frontendлҠ” backendлҘј requestsлЎң нҳём¶ңн•ҳкё°л§Ң н•ңлӢӨ.

frontend.py
import os
import requests
from flask import Flask

app = Flask(__name__)
#backend нҳём¶ң лҢҖмғҒ (compose л„ӨнҠёмӣҢнҒ¬мқҳ м»Ён…Ңмқҙл„Ҳ мқҙлҰ„мңјлЎң м ‘к·ј)
BACKEND = os.environ.get("BACKEND_URL", "http://otel-backend:9000/api")


@app.route("/")
def index():
    #requests мһҗлҸҷ кі„мёЎмқҙ traceparent н—ӨлҚ”лҘј мӢӨм–ҙ backendлЎң м „нҢҢ
    r = requests.get(BACKEND, timeout=5)
    return {"service": "dotoryeee-frontend", "backend_response": r.json()}

backendлҠ” л°ӣмқҖ мҡ”мІӯмқҳ traceparent н—ӨлҚ”лҘј лЎңк·ёлЎң м°ҚлҠ”лӢӨ. мқҙ н•ң мӨ„мқҙ л’Өм—җм„ң м»Ён…ҚмҠӨнҠё м „нҢҢлҘј мңЎм•ҲмңјлЎң нҷ•мқён•ҳлҠ” к·јкұ°к°Җ лҗңлӢӨ.

backend.py
from flask import Flask, request

app = Flask(__name__)


@app.route("/api")
def api():
    #frontendк°Җ м „нҢҢн•ң traceparent н—ӨлҚ”лҘј лЎңк·ёлЎң лӮЁкёҙлӢӨ
    traceparent = request.headers.get("traceparent")
    print(f"[dotoryeee-backend] received traceparent={traceparent}", flush=True)
    return {"service": "dotoryeee-backend", "message": "hello from dotoryeee-backend"}

zero-code кі„мёЎмқҳ н•өмӢ¬мқҖ мқҙлҜём§Җ л№Ңл“ң лӢЁкі„м—җ мһҲлӢӨ. opentelemetry-distroлҘј м„Өм№ҳн•ң л’Ө opentelemetry-bootstrapмқ„ лҸҢлҰ¬л©ҙ, мқҙлҜё к№”лҰ° лқјмқҙлёҢлҹ¬лҰ¬(Flask, requests)лҘј к°җм§Җн•ҙ к·ём—җ л§һлҠ” мһҗлҸҷ кі„мёЎ нҢЁнӮӨм§ҖлҘј м•Ңм•„м„ң м„Өм№ҳн•ңлӢӨ.

Dockerfile
FROM python:3.12-slim
WORKDIR /app

#opentelemetry-bootstrapмқҙ pkg_resourcesлҘј м°ёмЎ°н•ҳлҜҖлЎң setuptools нҸ¬н•Ё
RUN pip install --no-cache-dir \
    setuptools==75.1.0 \
    flask==3.0.3 \
    requests==2.32.3 \
    opentelemetry-distro==0.48b0 \
    opentelemetry-exporter-otlp==1.27.0

#м„Өм№ҳлҗң лқјмқҙлёҢлҹ¬лҰ¬м—җ л§һлҠ” мһҗлҸҷ кі„мёЎ нҢЁнӮӨм§ҖлҘј к°җм§Җн•ҙ м„Өм№ҳ -> мҪ”л“ң мҲҳм • м—Ҷмқҙ кі„мёЎ
RUN opentelemetry-bootstrap -a install

COPY frontend.py backend.py /app/

Warning

рҹ’Ў python:3.12-slimмқҖ setuptoolsлҘј л№јкі  л°°нҸ¬н•ҳлҜҖлЎң opentelemetry-bootstrap мӢӨн–ү м „м—җ setuptoolsлҘј н•Ёк»ҳ м„Өм№ҳн•ңлӢӨ

мӢӨн–үн•  л•ҢлҠ” flask лӘ…л № м•һм—җ opentelemetry-instrumentлҘј л¶ҷмқёлӢӨ. мқҙ лһҳнҚјк°Җ н”„лЎңм„ёмҠӨлҘј к°җмӢё кі„мёЎмқ„ мЈјмһ…н•ҳкі , OTEL_лЎң мӢңмһ‘н•ҳлҠ” нҷҳкІҪліҖмҲҳлЎң м„ңл№„мҠӨ мқҙлҰ„кіј export лҢҖмғҒмқ„ л„ҳкёҙлӢӨ.

composeлЎң мҠӨнғқ кё°лҸҷ

л„Ө м„ңл№„мҠӨлҘј н•ң нҢҢмқјлЎң л¬¶лҠ”лӢӨ. м•ұ м»Ён…Ңмқҙл„ҲлҠ” к°ҷмқҖ мқҙлҜём§ҖлҘј м“°кі  commandмҷҖ OTEL_SERVICE_NAMEл§Ң лӢӨлҘҙкІҢ мӨҖлӢӨ. Jaeger UIм—җлҠ” мқҙ м„ңл№„мҠӨ мқҙлҰ„мқҙ к·ёлҢҖлЎң л…ём¶ңлҗҳлҜҖлЎң dotoryeee-frontend, dotoryeee-backendлЎң м§“лҠ”лӢӨ.

л‘җ м•ұмқҙ кіөмң н•ҳлҠ” OTEL export м„Өм •мқҖ YAML м•өм»ӨлЎң н•ң лІҲл§Ң м Ғкі  мһ¬мӮ¬мҡ©н•ңлӢӨ.

docker-compose.yml
x-otel-env: &otel_env          #л‘җ м•ұмқҙ кіөмң н•ҳлҠ” export м„Өм •
  OTEL_EXPORTER_OTLP_ENDPOINT: http://otel-collector:4317
  OTEL_EXPORTER_OTLP_PROTOCOL: grpc
  OTEL_TRACES_EXPORTER: otlp
  OTEL_METRICS_EXPORTER: none  #мқҙ мӢӨмҠөмқҖ traceл§Ң лӢӨлЈЁлҜҖлЎң лҒ”
  OTEL_LOGS_EXPORTER: none

services:
  otel-collector:
    image: otel/opentelemetry-collector-contrib:0.109.0
    container_name: otel-collector
    command: ["--config=/etc/otelcol/config.yaml"]
    volumes: ["./otel-collector-config.yaml:/etc/otelcol/config.yaml:ro"]
    ports: ["4317:4317", "4318:4318"]
    depends_on: [otel-jaeger]

  otel-jaeger:
    image: jaegertracing/all-in-one:1.62.0
    container_name: otel-jaeger
    environment: [COLLECTOR_OTLP_ENABLED=true]
    ports: ["16686:16686"]

  otel-backend:
    build: ./app
    image: otel-lab-app:local
    container_name: otel-backend
    command: opentelemetry-instrument flask --app backend run --host 0.0.0.0 --port 9000
    environment: { <<: *otel_env, OTEL_SERVICE_NAME: dotoryeee-backend }
    depends_on: [otel-collector]

  otel-frontend:
    image: otel-lab-app:local
    container_name: otel-frontend
    command: opentelemetry-instrument flask --app frontend run --host 0.0.0.0 --port 8000
    environment:
      <<: *otel_env
      OTEL_SERVICE_NAME: dotoryeee-frontend
      BACKEND_URL: http://otel-backend:9000/api
    ports: ["18000:8000"]
    depends_on: [otel-backend, otel-collector]
  • нҳёмҠӨнҠё 8000 нҸ¬нҠёк°Җ мқҙлҜё м“°мқҙкі  мһҲм–ҙ frontendлҠ” 18000мңјлЎң л§Өн•‘н–ҲлӢӨ
  • CollectorмҷҖ JaegerлҠ” 4317мқ„ к°Ғмһҗмқҳ м»Ён…Ңмқҙл„Ҳ м•Ҳм—җм„ң м“°лҜҖлЎң нҸ¬нҠёк°Җ кІ№м№ҳм§Җ м•ҠлҠ”лӢӨ

debug exporterлҠ” span н•ҳлӮҳн•ҳлӮҳлҘј detailedлЎң н’Җм–ҙ м°ҚлҠ”лӢӨ. frontendмқҳ лЈЁнҠё spanмқ„ ліҙл©ҙ service.nameмқҙ dotoryeee-frontendлЎң мұ„мӣҢм ё мһҲкі , HTTP мҶҚм„ұмқҙ мһҗлҸҷ кі„мёЎмңјлЎң л¶ҷм–ҙ мһҲлӢӨ. л§Ё м•„лһҳ dotoryeee.labмқҖ Collectorмқҳ attributes processorк°Җ мЈјмһ…н•ң нғңк·ёлӢӨ.

  • http.method, http.status_code к°ҷмқҖ мҶҚм„ұмқҖ Flask мһҗлҸҷ кі„мёЎмқҙ мӢңл§ЁнӢұ м»ЁлІӨм…ҳм—җ л§һм¶° мұ„мҡҙлӢӨ
  • dotoryeee.labмқҖ м•ұмқҙ м•„лӢҲлқј Collectorк°Җ л¶ҷмҳҖлӢӨ вҶ’ processorк°Җ нҢҢмқҙн”„лқјмқё мӨ‘к°„м—җм„ң spanмқ„ мӢӨм ңлЎң мҶҗлҢҖкі  мһҲлӢӨлҠ” мҰқкұ°

л¶„мӮ° нҠёл ҲмқҙмҠӨ нҷ•мқё

Jaeger UI м—Ҷмқҙ HTTP APIлЎң лЁјм Җ кө¬мЎ°лҘј лңҜм–ҙліёлӢӨ. л“ұлЎқлҗң м„ңл№„мҠӨлҘј мЎ°нҡҢн•ҳл©ҙ л‘җ м„ңл№„мҠӨк°Җ мһЎнһҢлӢӨ.

curl -s http://localhost:16686/api/services
{"data":["dotoryeee-frontend","dotoryeee-backend"],"total":2,"limit":0,"offset":0,"errors":null}

frontendлЎң л“Өм–ҙмҳЁ trace лӘ©лЎқмқ„ лҪ‘м•„ traceлі„ span мҲҳмҷҖ м°ём—¬ м„ңл№„мҠӨлҘј м •лҰ¬н•ңлӢӨ. м„ё лІҲ нҳём¶ңн•ң л§ҢнҒј trace м„ё к°ңк°Җ мһЎнһҲкі , к°Ғ traceк°Җ л‘җ м„ңл№„мҠӨм—җ кұёміҗ span м„ё к°ңлЎң кө¬м„ұлҗңлӢӨ.

curl -s 'http://localhost:16686/api/traces?service=dotoryeee-frontend&limit=20&lookback=2h' \
 | jq -r '.data[] | "\(.traceID[0:8])  spans=\(.spans|length)  services=\([.processes[].serviceName]|sort|join(","))"'

trace н•ҳлӮҳлҘј кіЁлқј span нҠёлҰ¬лҘј мӢңмһ‘ мӢңк°Ғ мҲңмңјлЎң нҺјм№ңлӢӨ. мҡ”мІӯ н•ҳлӮҳк°Җ м„ңлІ„ spanм—җм„ң мӢңмһ‘н•ҙ нҒҙлқјмқҙм–ёнҠё spanмңјлЎң лӮҙл Өк°Җкі , к·ё м•„лһҳм—җ backendмқҳ м„ңлІ„ spanмқҙ л¶ҷлҠ”лӢӨ.

  • frontend GET /: мӮ¬мҡ©мһҗ мҡ”мІӯмқ„ л°ӣмқҖ лЈЁнҠё м„ңлІ„ span, л¶ҖлӘЁк°Җ м—ҶлӢӨ
  • frontend GET: requestsк°Җ backendлҘј л¶ҖлҘҙл©° л§Ңл“  нҒҙлқјмқҙм–ёнҠё span
  • backend GET /api: к·ё нҳём¶ңмқ„ л°ӣмқҖ backendмқҳ м„ңлІ„ span

к°ҷмқҖ кө¬мЎ°лҘј Jaeger UIм—җм„ңлҸ„ ліёлӢӨ. м„ңл№„мҠӨлЎң dotoryeee-frontendлҘј кі лҘҙл©ҙ trace м„ё кұҙмқҙ лңЁкі , к°Ғ н–үм—җ 3 SpansмҷҖ dotoryeee-backend, dotoryeee-frontendк°Җ н•Ёк»ҳ н‘ңмӢңлҗңлӢӨ.

trace н•ҳлӮҳлҘј м—ҙл©ҙ span кі„мёөмқҙ к·ёлҢҖлЎң л“ңлҹ¬лӮңлӢӨ. dotoryeee-frontendмқҳ GET / м•„лһҳ GET нҒҙлқјмқҙм–ёнҠё spanмқҙ, лӢӨмӢң к·ё м•„лһҳ dotoryeee-backendмқҳ GET /apiк°Җ мӨ‘мІ©лҗҳкі , мғҒлӢЁ мҡ”м•Ҫм—җ Services 2, Total Spans 3мңјлЎң мһЎнһҢлӢӨ.

н•ң н”„лЎңм„ёмҠӨм—җм„ң мӢңмһ‘н•ң мҡ”мІӯмқҙ лӢӨлҘё н”„лЎңм„ёмҠӨмқҳ мһ‘м—…к№Ңм§Җ н•ҳлӮҳмқҳ traceлЎң л¬¶мҳҖлӢӨ. мқҙкұё к°ҖлҠҘн•ҳкІҢ н•ҳлҠ” кІҢ лӢӨмқҢ м Ҳмқҳ м»Ён…ҚмҠӨнҠё м „нҢҢлӢӨ.

м»Ён…ҚмҠӨнҠё м „нҢҢ нҷ•мқё

л‘җ м„ңл№„мҠӨлҠ” к°ҷмқҖ н”„лЎңм„ёмҠӨк°Җ м•„лӢҲлӢӨ. к·ёлҹ°лҚ°лҸ„ spanмқҙ н•ң traceлЎң л¬¶мқё кұҙ frontendк°Җ HTTP мҡ”мІӯм—җ trace мӢқлі„мһҗлҘј мӢӨм–ҙ ліҙлғҲкё° л•Ңл¬ёмқҙлӢӨ. backendк°Җ л°ӣмқҖ н—ӨлҚ”лҘј м°Қм–ҙл‘” лЎңк·ёлҘј ліёлӢӨ.

traceparentлҠ” W3C Trace Context н‘ңмӨҖ н—ӨлҚ”мқҙкі , 00-{trace-id}-{parent-span-id}-01 нҳ•мӢқмқҙлӢӨ. frontendк°Җ backendлҘј л¶ҖлҘј л•Ң requests мһҗлҸҷ кі„мёЎмқҙ мқҙ н—ӨлҚ”лҘј мӢ¬лҠ”лӢӨ. мІ« мӨ„мқ„ м•һм„  debug м¶ңл Ҙкіј л§һм¶°ліҙл©ҙ кҙҖкі„к°Җ м •нҷ•нһҲ л“Өм–ҙл§һлҠ”лӢӨ.

traceparent мЎ°к°Ғ к°’ debug м¶ңл Ҙм—җм„ңмқҳ м •мІҙ
trace-id 63e823ab...f75e frontend, backend spanмқҙ кіөмң н•ҳлҠ” trace ID
parent-span-id c469107a75aafbd5 frontendмқҳ GET нҒҙлқјмқҙм–ёнҠё span ID

мҰү frontendк°Җ ліҙлӮё traceparentмқҳ parent-span-idлҠ” backend м„ңлІ„ spanмқҳ Parent IDк°Җ лҗңлӢӨ. backendлҠ” мқҙ н—ӨлҚ”лҘј мқҪм–ҙ мғҲ spanмқҳ л¶ҖлӘЁлЎң мӮјм•ҳкі , к·ёлһҳм„ң л‘җ н”„лЎңм„ёмҠӨмқҳ spanмқҙ н•ң лӮҳл¬ҙм—җ л¶ҷм—ҲлӢӨ. н—ӨлҚ”лҘј м§Ғм ‘ л§Ңл“Өкұ°лӮҳ нҢҢмӢұн•ҳлҠ” мҪ”л“ңлҠ” м•ұ м–ҙл””м—җлҸ„ м—ҶлӢӨ вҶ’ м „нҢҢлҸ„ мһҗлҸҷ кі„мёЎмқҙ мІҳлҰ¬н•ңлӢӨ.

processorлЎң span к°Җкіө

м•һм—җм„ң debug м¶ңл Ҙм—җ м°ҚнһҢ dotoryeee.labмқҙ attributes processorмқҳ кІ°кіјлӢӨ. м„Өм •м—җм„ң мқҙ processorлҘј нҢҢмқҙн”„лқјмқём—җ л„ЈмқҖ кІғл§ҢмңјлЎң лӘЁл“  traceмқҳ лӘЁл“  spanм—җ нғңк·ёк°Җ л¶ҷм—ҲлӢӨ. м•ұмқ„ лӢӨмӢң л°°нҸ¬н•  н•„мҡ”к°Җ м—ҶлӢӨлҠ” кІҢ Collector к°Җкіөмқҳ мһҘм җмқҙлӢӨ.

м—җлҹ¬лӮҳ лҠҗлҰ° мҡ”мІӯл§Ң лӮЁкё°кі  мӢ¶мңјл©ҙ tail мғҳн”Ңл§Ғмқ„ м–№лҠ”лӢӨ. tail_samplingмқҖ traceк°Җ лҒқлӮ  л•Ңк№Ңм§Җ spanмқ„ лІ„нҚјл§Ғн•ң л’Ө мЎ°кұҙмңјлЎң кұёлҹ¬лӮҙлҜҖлЎң, н•ң traceмқҳ лӘЁл“  spanмқҙ к°ҷмқҖ CollectorлЎң лӘЁм—¬м•ј м„ұлҰҪн•ңлӢӨ. м•„лһҳлҠ” мӢӨн–үк№Ңм§ҖлҠ” н•ҳм§Җ м•ҠмқҖ м„Өм • мҳҲмӢңлӢӨ.

#м„Өм • мҳҲмӢң (ліё мӢӨмҠөм—җм„ңлҠ” лҜём Ғмҡ©)
processors:
  tail_sampling:
    decision_wait: 10s          #trace мҷ„м„ұмқ„ кё°лӢӨлҰ¬лҠ” мӢңк°„
    policies:
      - name: errors            #м—җлҹ¬к°Җ нҸ¬н•Ёлҗң traceлҠ” л¬ҙмЎ°кұҙ ліҙмЎҙ
        type: status_code
        status_code: { status_codes: [ERROR] }
      - name: slow              #200ms мҙҲкіј м§Җм—° trace ліҙмЎҙ
        type: latency
        latency: { threshold_ms: 200 }

м •лҰ¬

мӢӨмҠөмқҙ лҒқлӮҳл©ҙ м»Ён…Ңмқҙл„ҲмҷҖ л„ӨнҠёмӣҢнҒ¬, ліјлҘЁмқ„ н•Ёк»ҳ лӮҙлҰ°лӢӨ. кё°мЎҙм—җ лҸҢлҚҳ лӢӨлҘё м»Ён…Ңмқҙл„ҲлҠ” кұҙл“ңлҰ¬м§Җ м•Ҡкі  мқҙ мҠӨнғқл§Ң м •лҰ¬лҗңлӢӨ.

docker compose down -v
  • мҪ”л“ң н•ң мӨ„ м•Ҳ кі м№ҳкі  Flask л‘җ м„ңл№„мҠӨм—җ кі„мёЎмқ„ мЈјмһ…н•ҙ л¶„мӮ° traceлҘј л§Ңл“Өм—ҲлӢӨ вҶ’ zero-code кі„мёЎмқҳ мӢӨмІҙлҠ” opentelemetry-instrument лһҳнҚјмҷҖ bootstrapмқҙ к№җ мһҗлҸҷ кі„мёЎ нҢЁнӮӨм§Җ
  • CollectorлҠ” л°ӣмқҖ spanмқ„ debugлЎң лҲҲм—җ ліҙм—¬мЈјкі , attributes processorлЎң к°Җкіөн•ң л’Ө JaegerлЎң л„ҳкІјлӢӨ вҶ’ мҲҳм§‘В·к°ҖкіөВ·м „лӢ¬мқҙ м•ұкіј л¶„лҰ¬лҗң н•ң кіім—җ лӘЁмқёлӢӨ
  • л‘җ н”„лЎңм„ёмҠӨмқҳ spanмқҙ н•ң traceлЎң л¬¶мқё к·јкұ°лҠ” traceparent н—ӨлҚ” н•ҳлӮҳмҳҖлӢӨ. л¶„мӮ° нҠёл ҲмқҙмӢұмқҖ кІ°көӯ "мқҙ н—ӨлҚ”лҘј л№ м§җм—Ҷмқҙ мқҙм–ҙмЈјлҠ” кІғ"мңјлЎң м••м¶•лҗңлӢӨ
